# Android IP Camera (夜视增强版)

[English](README.md) | **中文**

这是原版项目 [android-ip-camera](https://github.com/DigitallyRefined/android-ip-camera) 的一个修改分支。我根据个人需求加了一些功能，主要是为了让旧手机（比如我的三星 S10）能更好地在晚上或者暗光环境下当监控用。

***

## 增加的功能

### 增强暗光下的画面

加了一些算法来提高晚上的画面质量：

- **像素合并 (Pixel Binning)**：在手机端对原始数据做 2x2 平均处理，能有效减少高 ISO 下的噪点，让画面更干净。
- **多帧平均 (软件长曝光)**：在浏览器端把连续的几帧画面叠在一起。对静态场景（比如看宠物）效果很好，能像长曝光一样消除噪点。
- **对比度和亮度调节 (CLAHE & Gamma)**：能把阴影处的细节拉出来，同时防止背景太亮。
- **伪彩色 (热力图)**：把亮度映射成不同颜色。在肉眼看不清灰阶的时候，这种模式能帮你发现暗处细微的移动。
- **边缘增强**：一个简单的锐化滤镜，让物体的轮廓看起来更清晰一点。

### 更细致的硬件控制

手动控制摄像头的一些参数：

- **手动 ISO 和快门**：不用让系统自动调，在暗环境下可以锁定快门时间（比如 1/10s）来增加进光量。
- **手动对焦**：有个滑块可以手动调焦距。防止摄像头在黑夜里“拉风箱”到处找焦点。
- **镜头独立设置**：不同镜头（超广角、主摄、长焦）的设置会自动保存，切换的时候会自动恢复。
- **补光灯亮度**：如果你的设备支持（Android 13+），可以手动调物理闪光灯的亮度级别。

### 守宫模式 (运动监测)

专门为监测动作比较慢的目标设计的。它不是对比相邻的两帧，而是对比几秒钟之前的画面。这样即使是很慢的位移（比如守宫爬行）也能被抓到。

### 网页后台的一些改动

- **4K 支持**：对高分辨率串流的支持更好一些。
- **网络压力显示**：能实时看到当前的带宽占用，方便选合适的分辨率。
- **小地图**：缩放很大的时候，有个小地图能让你知道当前看的是哪块区域，还有些bug，但懒得修了。
- **照片画廊**：远程拍的高清照片可以在网页端的侧边栏直接查看。

## 如何开始

1. **基础安装**：关于怎么安装、怎么生成 HTTPS 证书等基础操作，请参考 [原项目仓库](https://github.com/DigitallyRefined/android-ip-camera)。
2. **使用建议**：打开网页后台，展开 **Night Enhancement** 面板，根据你的环境打开对应的增强算法就行。

## ⚠️ 注意

开启这些增强算法和高分辨率后，手机发热会比较明显：

- 如果你的手机有\*\*“电池保护 (限电至 80%)”\*\*功能，建议一定要开启。
- 建议多用 **像素合并 (Pixel Binning)**，能在保证画质的同时减轻传输和处理压力。
