# Android IP Camera (专业夜视增强版)

[English](README.md) | **中文**

这是一个针对原版 Android IP Camera 的深度定制分支，专为将旧款旗舰设备（如 **三星 S10**）转化为专业级**低光观测站**而设计。

原项目提供了稳健的串流基础，而本版本则完全聚焦于**极限夜视能力**、**硬件级精准控制**以及**智能生物监测（守宫模式）**。

---

## 🌙 “暗夜猎手”套件：极限低光功能

本版本实现了一套多级图像增强流水线，效果可媲美专用的数字夜视硬件：

### 🧬 手机端像素合并 (Pixel Binning)
- **原理**：在 JPEG 压缩前，直接在手机端对 **NV21 原始数据**进行 2x2 空间均值合并。
- **效果**：有效提升信噪比 (SNR)，从源头“粉碎”感光元件噪点。即使在极高 ISO 下，也能提供非常纯净的画面。

### ⏳ 帧平均多帧叠加 (软件长曝光)
- **原理**：在浏览器端利用递归加权平均算法堆叠 1-10 帧连续画面。
- **效果**：为静态场景（如宠物饲养箱）产生“长曝光”效果，几乎消除所有时间噪点，揭示近乎全黑环境下的物体。

### 🌓 高级色调映射与对比度
- **CLAHE (局部自适应对比度增强)**：将画面分为 8x8 网格进行局部对比度拉伸。这能揭示最暗阴影处（如躲避穴内部）的细节，同时防止背景强光过曝。
- **伽马校正 (Gamma Correction)**：非线性曲线调节，智能提升中暗部亮度。
- **数字增益 (Digital Gain)**：针对极端暗环境的非线性软件放大。

### 🌈 伪彩色 (热力图模拟)
- 将 0-255 的亮度范围映射到“蓝 -> 绿 -> 黄 -> 红”的色谱。
- 极其适合在肉眼无法分辨灰度阶的极暗环境中，识别微弱的热特征或移动模式。

### 📐 边缘增强 (锐化)
- 使用 3x3 卷积核恢复在长曝光或高 ISO 模式下变得“模糊”的物体轮廓。

## 🎯 专业硬件精准控制

全面接管设备的镜头光学系统：

- **镜头独立设置持久化**：系统会自动为每个镜头（超广角、主摄、长焦）独立保存并恢复设置（ISO、快门、对焦、Binning）。
- **全手动曝光 (ISO & 快门)**：直接覆盖 Camera2 API。可锁定快门至 1/10s 或将 ISO 提升至 3200+ 以最大化光子收集。
- **手动对焦距离**：精准滑块控制（0 = 无穷远 到 微距），防止在黑暗中出现“对焦拉风箱”。
- **🎯 强制自动对焦**：远程一键触发硬件对焦/测光循环。
- **闪光灯亮度调节 (API 33+)**：精细控制物理补光灯的亮度级别。

---

## 🦎 守宫模式 (Gecko Mode)：智能运动监测

专为监测移动缓慢的爬行动物或低活跃度目标设计：

- **慢速模式 (Gecko Mode)**：将当前帧与最长 **5 秒前**的参考帧进行对比。这能捕捉到传统“相邻帧”算法无法识别的极慢位移。
- **后台持续运行**：利用 **Screen Wake Lock API** 和直接数据层处理，确保即使浏览器标签页在后台或最小化，监测和音频报警依然有效。
- **像素强度微调**：可调的“变动”阈值，让你在过滤感光噪点的同时，精准捕捉守宫的移动。

---

## 🛜 现代化远程仪表盘

- **4K 超高清支持**：硬件自动探测 4K 串流，并配合智能网络背压管理。
- **网络压力指示器**：实时 Mbps 监测与状态点（绿/黄/红），助你选择最合适的传输分辨率。
- **可拖动缩放小地图**：针对高达 8 倍的数字变焦，提供精准的视口定位。
- **高精抓拍画廊**：远程触发传感器全像素拍照，并在可折叠侧边栏画廊中管理高清大图。

## ⚠️ 稳定性与安全提示

在开启多项增强算法和高分辨率模式下 24/7 运行会产生显著热量：
- 务必开启 Android 设备的 **“电池保护 (限电至 80%)”** 功能。
- 建议优先使用 **像素合并 (Pixel Binning)** 技术，在保持高信号质量的同时降低传输带宽和处理压力。
